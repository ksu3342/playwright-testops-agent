# Demo Walkthrough

Suggested demo flow:

1. Show the sample input in `data/inputs/sample_prd_login.md`
2. Run `python -m app.main parse --input data/inputs/sample_prd_login.md`
3. Run `python -m app.main generate --input data/inputs/sample_prd_login.md`
4. Run `python -m app.main run --input data/inputs/sample_prd_login.md`
5. Run `python -m app.main report --run-id latest --input data/inputs/sample_prd_login.md`
6. Open the generated files in `generated/tests/`, `generated/reports/`, and `data/runs/`

Talking point:
This is a truthful scaffold, not a fake end-to-end autonomous platform.
